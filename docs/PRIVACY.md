# Privacy and Safety

OpenMage AI Profiler is designed to generate reports that can be safely shared with developers, consultants or AI assistants.

## Excluded Data

The profiler must not collect:

- Customer records
- Orders
- Quotes
- Admin users
- Passwords
- Crypt keys
- API keys
- Payment credentials
- Personal data
- Email addresses
- Telephone numbers

## Redaction

Sensitive configuration values must be masked.

Examples:

- password
- secret
- token
- key
- api
- private
- payment
- stripe
- paypal
- sagepay
- auth
- credential
