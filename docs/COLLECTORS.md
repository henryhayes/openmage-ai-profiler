# Collectors

Collectors are responsible for gathering specific categories of information from a Magento 1.x or OpenMage installation.

## Planned Collectors

- System
- PHP
- Magento
- Filesystem
- Stores
- Modules
- Themes
- Layouts
- Routers
- Controllers
- Blocks
- Models
- Helpers
- Events
- Observers
- Cron
- Cache
- Indexes
- Products
- Categories
- Attributes
- EAV
- CMS
- Database
- Security
- Code Analysis
- Rewrite Analysis
- Module Analysis
