## [0.8.0] - 2026-07-05

### Changed

- Automatic collector discovery now keeps a preferred, AI-friendly report order while still enforcing declared collector dependencies.
- ChatGPT prompt output is now instructions-only and no longer duplicates the compact AI context.
- Updated project version to 0.8.0.

### Fixed

- Cache collector now reports session storage from the same Magento global session configuration used by the Magento collector.

---

## [0.7.0] - 2026-07-05

### Added

- EAV collector.
- Layout collector.
- Router collector.
- Controller collector.
- Automatic collector directory registration.
- Collector dependency sorting.
- AI context summaries for EAV, layout, router and controller architecture.

---

## [0.6.0] - 2026-07-05

### Added

- Index collector.
- Cache collector.
- Database collector.
- Rewrite map collector.
- AI context summaries for indexes, cache, database and rewrite map.

### Changed

- ChatGPT prompt output now recommends TXT and JSON reports as the primary detailed profile files.
- Performance guidance now references cache, index and cron architecture.
- Database guidance now references database architecture.

---

## [0.5.0] - 2026-07-05

### Added

- AI context writer.
- ChatGPT prompt writer.
- Report manager.
- Observer collector.
- Cron collector.
- Optional Markdown output via `--markdown`.

### Changed

- Report writing is now handled by `ReportManager`.
- Markdown output is no longer generated by default.

---

## [0.4.0] - 2026-07-05

### Added

- Collector framework
- Environment collector
- PHP collector
- Magento bootstrap collector
- Magento collector
- Store collector
- Module collector
- Theme collector
- Theme hierarchy collector
- Rewrite collector
- ResourceLocator helper
- Filesystem helper
- XmlHelper
- TXT report writer
- Markdown report writer
- JSON report writer

### Changed

- Reports are now grouped by collector category.
- Theme analysis now understands Magento fallback hierarchy.
- Module analysis now includes filesystem and XML inspection.

### Fixed

- Improved Magento bootstrap detection.
- Improved report structure and collector metadata.
